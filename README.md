# 🤖 AgentulMeu.online — Configurează-ți Agentul AI în 8 Pași

> **Primul sistem din România care generează automat identitatea completă a agentului tău AI — personalizat pe business-ul tău.**

---

## Ce este AgentulMeu?

AgentulMeu este o platformă web care transformă un interviu structurat de 8 pași într-un sistem complet de fișiere de configurare pentru agentul tău AI personal.

Nu ai nevoie de programatori. Nu ai nevoie de cunoștințe tehnice. Completezi intake-ul, apeși un buton, și primești **9 fișiere MD gata de utilizat** în orice sistem AI: OpenClaw, Claude, GPT, Gemini sau orice LLM.

### Pasul 2 — Identitate & context

Informațiile de bază din care agentul construiește înțelegerea completă a contextului.

**Câmpuri (toate cu AI assist ✨):**
- Numele business-ului / brandului
- Ce face business-ul în 2 propoziții (specific, nu generic)
- Produse / servicii principale cu prețuri
- Clientul ideal — demografic, comportament, buget, problemă
- Zona geografică / piața acoperită
- Website / social media principale
- Modelul de venit

### Pasul 4 — Echipa de agenți

Selectează agenții care au sens pentru obiectivele tale. Fiecare adaugă o capabilitate distinctă.

> **Recomandare:** Începe cu 3–4 agenți și scalează. Mai puțini agenți bine configurați > mulți agenți mediocri.

**Agenți de execuție:**

| Agent | Rol |
|-------|-----|
| 🎯 HUNTER | Caută prospecți pe LinkedIn, Google Maps, forumuri. Scor calificare automată |
| ✍️ WRITER | Generează postări, articole, emailuri, scripturi video. Ton adaptat brandului |
| 📞 CLOSER | Follow-up secvențial, răspunde lead-uri inbound, pregătește oferte personalizate |
| 🛎️ SUPPORT | Răspunde FAQ, gestionează reclamații, onboarding clienți noi |
| 📊 ANALYST | Rapoarte KPI, forecasting, alerte la anomalii |
| 🔭 SCOUT | Monitorizează competitori și piață. Briefing zilnic |

**Agenți specializați:**

| Agent | Rol |
|-------|-----|
| 🗓️ SCHEDULER | Programări, confirmări, remindere, sincronizare calendar |
| 🧾 FINANCE | Facturi, urmărire plăți, cashflow, alerte restanțe |
| 📦 OPS | Comenzi, stocuri, relație furnizori, logistică internă |
| 🧪 R&D | Studii, brevete, tehnologii noi, validare ipoteze |
| 🎓 TRAINER | Training, onboarding angajați, FAQ intern, wiki |
| 🏗️ PM | Taskuri, deadline-uri, blocaje, rapoarte de proiect |

Integrările determină cum agentul comunică și ce date poate accesa.

### Pasul 6 — Personalitate & reguli

Tonul și restricțiile sunt critice — agentul le respectă absolut în toate interacțiunile.

**Niveluri de autonomie:**

| Nivel | Descriere |
|-------|-----------|
| ✋ Aprobare pentru orice | Agentul pregătește, tu decizi și trimiți. Recomandat la start |
| ⚡ Semi-autonom | Rapoarte și drafturi automat. Mesaje externe cu aprobare |
| 🤖 Full autonom | Execută complet, primești raport zilnic. Doar pentru procese definite |

**Câmpuri cu AI assist:** Ton brand · Limbă · Program de lucru & notificări · Linii roșii absolute (reguli în SOUL.md) · Competitori (pentru SCOUT) · Avantaj competitiv real

### Pasul 7 — Context tehnic

Configurarea se adaptează la nivelul tehnic al utilizatorului.

**Niveluri experiență:**

| Nivel | Profil |
|-------|--------|
| 🌱 Începător | Am folosit ChatGPT casual, nu am automatizat nimic serios |
| ⚙️ Intermediar | Folosesc Make/Zapier, am setat chatboți simpli, înțeleg API-uri |
| 🚀 Avansat | Lucrez cu API-uri și cod, am construit pipelines, configurez VPS |

**Implicare în setup:**

| Tip | Descriere |
|-----|-----------|
| 🙋 Independent | Vreau să înțeleg și să pot modifica singur pe termen lung |
| 🤝 Colaborare | Implicat în decizii, dar nu singurul responsabil tehnic |
| 🎁 Done-for-you | Văd rezultatele, nu intru în detalii tehnice |

**Câmpuri libere:** Deadline lansare · Așteptare primele 30 zile · Context și constrângeri specifice

## Arhitectură tehnică

```
[Browser — agentulmeu.online]
         ↓
   Nginx reverse proxy
         ↓
   index.html (single-page app, zero frameworks)
         ↓
   Gemini 2.5 Flash API (direct din browser)
         ↓
   9 × MD generation (prompts individuale specializate)
         ↓
[Download individual / Paste în sistem AI]
```

**Stack:** HTML + CSS + Vanilla JS · Google Gemini 2.5 Flash · DM Sans + Syne · Ubuntu 24.04 + Nginx
